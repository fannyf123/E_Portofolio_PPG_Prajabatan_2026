## 📋 Komponen E-Portfolio 1

Website ini dirancang sesuai **Rubrik E-Portfolio 1 PPG Prajabatan**, mencakup 5 komponen utama:

| No | Komponen | Deskripsi |
|----|----------|-----------|
| 1 | **Profil Mahasiswa** | Narasi asal daerah, inspirasi menjadi guru, tujuan profesional, dan kutipan penguat profil |
| 2 | **Artefak Produk** | RPP (3 siklus), Modul Ajar, Media Pembelajaran, Contoh Hasil Kerja Siswa, Video, Penilaian GP & DPL |
| 3 | **Analisis Artefak** | Konteks, tujuan, kelebihan, kekurangan, kajian teori, dan integrasi PPL Terbimbing |
| 4 | **Lampiran Penilaian** | Lampiran 7 & 8 — Instrumen Penilaian Penyusunan Perangkat & Praktik Mengajar (3 siklus) |
| 5 | **Model Guru yang Dituju** | Visi & misi, kompetensi yang dikembangkan, karakter guru profesional, dan roadmap |

## ✨ Fitur Website

- 🎨 **Desain Modern** — Tampilan profesional dengan animasi partikel interaktif
- 🌙 **Dark / Light Mode** — Toggle tema gelap dan terang
- 📱 **Responsif** — Optimal di desktop, tablet, dan mobile
- 🔍 **Filter Artefak** — Saring artefak berdasarkan kategori (RPP, Modul, Media, Asesmen, dll)
- 📋 **Modal Analisis** — Popup detail analisis setiap artefak (konteks, tujuan, kelebihan, kekurangan, kajian teori)
- 📊 **Tabel Penilaian** — Lampiran 7 & 8 per siklus dari Guru Pamong
- 🗺️ **Roadmap Guru** — Visualisasi perjalanan menuju guru profesional

---

## 🛠️ Teknologi

- **HTML5** — Struktur halaman
- **CSS3** — Styling dengan custom properties, animasi, glassmorphism
- **Vanilla JavaScript** — Interaktivitas tanpa framework
- **Google Fonts** — Tipografi: Outfit + Inter
- **Canvas API** — Animasi partikel latar

## 📌 Cara Penggunaan

1. Clone atau download repository ini:
   ```bash
   git clone https://github.com/fannyf123/E_Portofolio_PPG_Prajabatan_2026.git
   ```
2. Buka file `index.html` di browser (tidak memerlukan web server)
3. Edit konten di `index.html` sesuai data asli Anda
4. Sesuaikan style per-section di folder `css/` (setiap file berisi satu section)
5. Sesuaikan logika JavaScript di folder `js/` sesuai kebutuhan
6. Upload ulang ke GitHub Pages atau hosting lainnya

### 🔧 Pengembangan Lokal

```bash
# Clone repository
git clone https://github.com/fannyf123/E_Portofolio_PPG_Prajabatan_2026.git
cd E_Portofolio_PPG_Prajabatan_2026

# Buka langsung di browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```
